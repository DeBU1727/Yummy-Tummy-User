# 🍔 Yummy-Tummy

Welcome to **Yummy-Tummy**, a full-stack sit-dine and food delivery web application. This platform allows users to seamlessly browse a curated menu, manage their cart, and securely place orders using a modern OTP-based authentication system.

## ✨ Key Features

* **Guest Browsing:** Anyone can visit the app, browse the limited menu, and add delicious items to their cart without needing to create an account first.
* **Secure OTP Authentication:** To finalize a purchase, users must log in. We use a secure, passwordless OTP (One-Time Password) system via email. 
* **Reliable Email Delivery:** OTP emails are powered by **Brevo** to ensure safe, non-spam delivery. *(Note: If you do not see the OTP in your inbox, please quickly check your spam/junk folder just in case).*
* **Simulated Checkout:** Built for educational purposes, the application features a simulated payment gateway. You can experience the full checkout flow without using real money or actual credit card details.
* **Responsive UI:** A clean, user-friendly interface built for both desktop and mobile browsing.

## 🛠️ Tech Stack
