🍔 Yummy-Tummy

A Sit-Dine & Food Delivery Web Application

📌 Overview

Yummy-Tummy is a web-based application designed for both sit-dine and food delivery services. It provides a smooth and secure user experience where customers can explore menu items, add them to the cart, and complete purchases using a simple OTP-based login system.

🚀 Features
🛒 Guest Access
Users can browse menu items without logging in
Add products to the cart as a guest
🔐 OTP-Based Authentication
Login is required only at checkout
Secure email OTP verification (no passwords needed)
Quick and user-friendly authentication system
📧 Secure Email Service
OTP emails are sent via Brevo
Reliable delivery and designed to avoid spam issues
🍽️ Limited Menu System
Curated and limited food menu
Easy navigation and faster decision-making
💳 Simulated Payment System
Payment process is simulated for educational purposes only
No real transactions are processed
🔑 Authentication Flow
User enters email at checkout
OTP is sent via Brevo
User verifies OTP
Login successful
Order can be placed
⚠️ Important Notes
📩 If you don’t receive the OTP email:
Check your Spam/Junk folder first
Verify your email address is correct
